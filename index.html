Projetos em Destaque

De ideias abstratas individuais a projetos em grupo e extracurriculares, aqui estão alguns dos meus trabalhos.

  • WeMovie

    Este projeto é um aplicativo mobile de e-commerce simplificado desenvolvido para o desafio de estruturação de projeto nativo Android da WeFit.

    O aplicativo foi construído utilizando a arquitetura MVVM (Model-View-View-Model), com Kotlin como linguagem de programação principal. Ele se conecta a uma API para exibir uma lista de filmes, permitindo que o usuário adicione itens ao carrinho e finalize a compra.



  • MediHelp

    (🚧Em Desenvolvimento🚧)

    A MediHelp é uma ferramenta para auxiliar médicos e enfermeiros no cálculo de doses de medicamentos para crianças, com base em peso, idade, altura e outros fatores.

    Nosso objetivo é permitir que os profissionais de saúde tenham mais eficiência em seu trabalho e mais tempo para o cuidado com os pacientes.



  • Virtual Menu

    O aplicativo contém duas telas: a primeira exibe uma lista de itens e a segunda fornece detalhes do item selecionado. Os dados são fixos, sem conexão com APIs externas.

    Desenvolvido utilizando o Android Studio e suas funcionalidades são: exibição de uma lista de itens de cardápio, detalhamento do item selecionado em uma segunda tela, navegação entre as telas de forma funcional. Layout responsivo via XML, além de utilizar itents e recyclerview para manipulação de elementos visuais.



  • Tela de Login

    Este projeto foi desenvolvido como parte do curso da Trybe, com o objetivo de criar a tela de login de uma rede social utilizando Kotlin. A tela contém apenas o layout e as validações de interface, sem se conectar a uma API.

    Este app android foi desenvolvido utilizando o Android Studio e suas funcionalidades são: Validação dos campos de login (e-mail e senha), layout responsivo e bem estruturado via XML e manipulação de elementos visuais utilizando Kotlin.



  • Site Hub Lar - Front End

    A Hub Lar é um negócio de impacto social de Sorocaba, que oferece serviços de arquitetura e construção civil para a população de baixa renda.

    Há nove anos, a Hub Lar é um escritório de arquitetura comprometido com o impacto social, dedicado a melhorar as condições de vida de famílias em situação de vulnerabilidade.



  • PIBTV Web Site

    (🚧Em Desenvolvimento🚧)


    O site está sendo desenvolvido com Next.js e TypeScript, e tem como objetivo ser um site informativo e interativo, onde os usuários podem encontrar informações sobre a igreja, eventos, cultos, fotos, vídeos e muito mais.
    Terá um sistema de login, onde os usuários poderão se cadastrar e acessar conteúdos exclusivos. A Equipe da igreja poderá gerenciar o conteúdo do site através de um painel de administração.



  • Desafio Be-Mobile Front-End

    O Teste Técnico para Front-End da Be consiste em construir a visualização de uma tabela com dados que virão de uma API simulada, em json-server. A tabela deve conter informações de usuários, como nome, cargo, data de admissão, telefone e imagem.

    Utilizei TypeScript com React.js para construir o projeto e CSS Puro para estilização. A aplicação é responsiva e contém um input de pesquisa que filtra os dados da tabela.



  • App de Receitas - Recipes App

    Um projeto feito durante a formação de desenvolvedor Full Stack na Trybe, o app de receitas é um projeto que utiliza React, Redux, Context API, Hooks e muito mais. Para facilitar e auxiliar na hora de cozinhar, o app conta com um timer, e um sistema de favoritos, onde o usuário pode salvar suas receitas favoritas. Seja um cozinheiro amador ou profissional, o app de receitas é para todos.



  • Password Manager

    O Password Manager é um projeto que utiliza React, Redux, Context API, Hooks e muito mais. O projeto consiste em um gerenciador de senhas, onde o usuário pode salvar suas senhas de forma segura e organizada. O Password Manager é um projeto que visa facilitar a vida do usuário, e garantir a segurança de suas senhas.



  • Pixels Art

    Nosso primeiro projeto na Trybe, o Pixels Art é um projeto que utiliza HTML, CSS e JavaScript. O projeto consiste em um quadro de pixels, onde o usuário pode escolher a cor e pintar os pixels, criando desenhos incríveis. O Pixels Art é um projeto que estimula a criatividade e a imaginação, e é para todas as idades.
